tests.sic.sST.det package: implements exact and/or heuristic algorithms for the global optimization of long-run exact average costs of a single-echelon inventory control system operating a periodic review control policy (R,nQ,T) facing deterministic demands with constant demand rate, in a continuous time cost-accounting setting.

Package Specification

The formulae for the cost of such a system are computed based on EOQ-type calculations by A.G. Lagodimos (2018).
