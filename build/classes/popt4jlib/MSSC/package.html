Package implements algorithms for Minimum-Sum-of-Square(-Errors)-Clustering, a well-defined optimization problem.

Package Specification

The ClustererIntf interface declares the methods that every MSSC clusterer should implement. The class GMeansMTClusterer is a thread-safe multi-threaded algorithm implementing the (hard-version) K-Means algorithm for MSSC clustering, without the possibility of multiple restarts. The K-Means++ method for seed initialization is also implemented.
